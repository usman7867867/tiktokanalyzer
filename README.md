# USMAN TikTok Analyzer

A production-ready Termux-based Python application that fetches publicly available TikTok profile information and generates detailed reports. Designed with a colorful menu, logging, configuration, and auto-update.

**Created By Muhammad Usman**

---

## Features

- **Username Analyzer** – Validate and scrape TikTok profiles (followers, bio, likes, etc.)
- **Report Generator** – Export results in JSON and TXT (timestamped)
- **Colorful Terminal UI** – Progress spinners, colored output
- **Auto‑update** – Check and pull updates from GitHub
- **Logging System** – All actions logged to `data/app.log`
- **Configuration** – Customizable via `config/settings.ini`
- **Security** – Input validation, safe file handling, exception management
